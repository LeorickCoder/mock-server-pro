# 变更日志

## [1.1.0] - 2023-03-24

### 新功能

- **参数化路由支持**: 添加了对Express风格参数化路由的完整支持
  - 实现了路径参数提取和注入
  - 支持复杂的路由模式，包括多参数、正则约束和通配符
  - 优化了路由匹配算法和性能
  - 添加了路由排序机制，确保更具体的路由先匹配
  - 添加了缓存机制，提高路由匹配性能
  - 完整文档和示例代码

### 优化

- **路由分发器重构**: 改进了MockDispatcher类
  - 使用数组存储路由定义，而不是简单的Map
  - 实现了基于模式匹配的路由查找算法
  - 添加了路由排序和缓存机制

### 开发工具

- **测试界面增强**: 扩展了HTTP测试页面
  - 添加了参数化路由测试界面
  - 支持测试各种类型的参数化路由

### 文档

- 添加了参数化路由实现文档 (`docs/path-params.md`)
- 更新了变更日志

## [1.0.0] - 2023-03-21

### 初始版本

- 基本的Mock服务器功能
- Express中间件支持
